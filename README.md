# odin-recipes
Este projeto foi desenvolvido como parte do meu aprendizado em desenvolvimento web, utilizando apenas HTML para estruturar uma página de receitas. A ideia foi criar um site simples, mas organizado, que apresentasse receitas de forma clara e visualmente agradável.

## ✨ Processo de Criação
Durante a construção da página, explorei diferentes elementos fundamentais do HTML:
- **Cabeçalhos (h1 a h3)**: para organizar o conteúdo em títulos principais e subtítulos, facilitando a hierarquia da informação.
- **Listas ul e ol**: usadas para apresentar ingredientes (listas não ordenadas) e etapas de preparo (listas ordenadas).
- **Imagens**: adicionadas para ilustrar as receitas e tornar a página mais atrativa.

## 📚 Aprendizados
- A importância da semântica no HTML, garantindo que cada elemento seja usado de forma correta e significativa.
- Como estruturar uma página de forma hierárquica, usando títulos e subtítulos para melhorar a leitura.
- A diferença entre listas ordenadas e não ordenadas, aplicando cada uma conforme o contexto.
- A utilização de imagens para enriquecer o conteúdo e melhorar a experiência do usuário.

## 🚀 Resultado
O resultado foi uma página simples, mas funcional, que apresenta receitas com:
- Título e descrição
- Lista de ingredientes
- Passo a passo organizado
- Imagens ilustrativas

Esse projeto foi um exercício prático que consolidou meus conhecimentos iniciais em HTML e me preparou para avançar em CSS e JavaScript, trazendo estilo e interatividade às próximas versões.
